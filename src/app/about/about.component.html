Welcome to the Serverless Blog

Purpose

I built this blog as a way to compile my Harvard projects and research in one place. While this is still a work in progress, the backend configuration featuring a serverless architecture is set-up. Over the next few weeks I hope to add more features.

Architecture

  • GraphQL - Data
  • AWS Amplify - Deployment
  • AWS Cognito - Authentication
  • AWS SES - Email Verification
  • AWS AppSync - Data and Realtime Updates
  • AWS CloudFront - Distribution
  • AWS S3 - Hosting
  • Angular - Frontend
